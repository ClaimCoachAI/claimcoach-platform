## For Contractors

### Getting Started

You will receive an email with a secure link to upload photos and submit a scope sheet for a property damage claim. This link is unique to you and expires after 7 days.

### Step 1: Access Your Magic Link

1. Check your email for a message from ClaimCoach
2. Click the "Upload Photos & Scope" button
3. You'll be taken directly to the upload page (no login required)

**Important:** Keep this link private. Anyone with the link can submit on your behalf.

### Step 2: Upload Photos

1. Click "Choose Files" or drag photos into the upload area
2. Upload photos showing:
   - Overall property views
   - Damaged areas (close-up shots)
   - Surrounding context
3. Add descriptions for each photo (optional but helpful)
4. Click "Next: Scope Sheet" when done

**Tips:**
- Use good lighting
- Take multiple angles of damage
- Include reference objects for scale
- Maximum 50 photos per claim

### Step 3: Fill Out the Scope Sheet

The scope sheet is a detailed form about the damage. You'll complete it in sections:

#### Section 1: Roof (Main)

Document the primary roof structure.

**Fields:**

- **Roof Type:** Select from dropdown (asphalt shingles, metal, tile, etc.)
- **Square Footage:** Total roof area in square feet
- **Pitch:** Roof slope (e.g., "6/12" for 6-inch rise per 12 inches)
- **Fascia:** Linear feet and whether it needs painting
- **Soffit:** Linear feet and whether it needs painting
- **Drip Edge:** Linear feet and painting
- **Pipe Jacks:** Count of plumbing vents and painting
- **Exhaust Vents:** Count and painting
- **Turbines:** Count and painting
- **Furnace Vents:** Count and painting
- **Power Vents:** Count and painting
- **Ridge Cap:** Linear feet
- **Satellites:** Count of satellite dishes
- **Step Flashing:** Linear feet (where roof meets wall)
- **Chimney Flashing:** Check if chimney needs flashing work
- **Rain Diverters:** Linear feet
- **Skylights:** Count and whether damaged

**Example:**

```
Roof Type: Asphalt Shingles
Square Footage: 2000
Pitch: 6/12
Fascia: 150 LF, Paint: Yes
Soffit: 100 LF, Paint: No
Drip Edge: 180 LF, Paint: No
Pipe Jacks: 4, Paint: No
Exhaust Vents: 2, Paint: Yes
```

#### Section 2: Roof (Other)

If the property has a secondary roof structure (e.g., garage, porch), document it here. Fields are the same as Section 1.

**Skip this section** if there's only one roof structure.

#### Section 3: Exterior

Document exterior damage by direction (Front, Right, Back, Left).

**For Each Side:**

- **Siding 1:** Square feet to replace and/or paint
- **Siding 2:** If multiple siding types, document second type
- **Gutters:** Linear feet and painting
- **Windows:** Count or description of damaged windows
- **Screens:** Count or description of damaged screens
- **Doors:** Description of damaged doors
- **AC Unit:** Check if replacement needed
- **AC Comb Fins:** Check if damaged

**Example (Front Side):**

```
Siding 1 Replace: 200 SF
Siding 1 Paint: 0 SF
Gutters: 40 LF, Paint: No
Windows: 3 windows damaged (bedroom, kitchen, living room)
Screens: 2 screens torn
AC Comb Fins: Yes
```

#### Section 4: Additional

Document any other damage not covered above.

- **Porch Paint:** Check if porch needs painting
- **Patio Paint:** Check if patio needs painting
- **Fence:** Describe fence damage (type, height, linear feet)
- **Additional Items (Main Roof):** Free text for other main roof items
- **Additional Items (Other Roof):** Free text for other roof items
- **Notes:** Any additional context or observations

**Example:**

```
Fence: Wood fence, 6ft tall, 100 LF damaged by fallen tree branch
Notes: Significant hail damage on north-facing roof slope. Multiple shingles completely missing. Damage appears to be from storm on January 15, 2026.
```

### Step 4: Review and Submit

1. Review all sections
2. Make sure measurements are accurate
3. Click "Submit Scope Sheet"
4. You'll see a confirmation message

**After Submission:**

- You cannot edit the scope sheet after submission
- The property manager will be notified
- You'll receive a confirmation email
- Your work is complete!

### Common Questions (Contractors)

**Q: How do I measure square footage?**

A: For roofs, measure length Ã— width of each section and add them together. Online calculators can help with complex roof shapes.

**Q: What if I don't know the pitch?**

A: Leave it blank. The property manager can fill it in later. If you have a pitch gauge, use it. Common pitches are 4/12, 6/12, 8/12.

**Q: Can I save my progress?**

A: Not currently. Try to complete the form in one session (takes 10-15 minutes).

**Q: What if I made a mistake?**

A: Contact the property manager who sent you the link. They can reset the scope sheet if needed.

**Q: My link expired. What now?**

A: Contact the property manager to request a new link.

## For Property Managers

### Overview

The AI Audit System helps you:
1. Collect detailed scope information from contractors
2. Generate industry-standard estimates
3. Compare carrier estimates to identify undervalued items
4. Create professional rebuttal letters

### Step 1: Send Magic Link to Contractor

1. Navigate to the claim details page
2. Click "Request Contractor Scope"
3. Enter contractor's email address
4. Click "Send Magic Link"

**What Happens:**

- Contractor receives email with secure link
- Link expires in 7 days
- You can resend if needed
- You'll be notified when submitted

### Step 2: Review Submitted Scope Sheet

When the contractor submits the scope sheet, you'll receive a notification.

**To Review:**

1. Open the claim
2. Click "Scope Sheet" tab
3. Review all submitted data
4. Verify measurements are reasonable

**Red Flags:**

- Unrealistic square footage (e.g., 10,000 SF for a house)
- Missing critical fields (roof type, square footage)
- Inconsistent data (more vents than possible)

**Actions:**

- **Accept:** Proceed to generate estimate
- **Request Changes:** Send back to contractor with notes
- **Edit:** Make minor corrections yourself

### Step 3: Generate Industry Estimate

Once you're satisfied with the scope sheet:

1. Click "Generate Industry Estimate"
2. System sends scope data to AI (takes 30-60 seconds)
3. Review the line-by-line estimate

**What You'll See:**

```
Industry-Standard Estimate

Roofing
- Remove existing asphalt shingles: 2000 SF @ $2.50 = $5,000.00
- Install new asphalt shingles: 2000 SF @ $4.50 = $9,000.00
- Install drip edge: 180 LF @ $2.50 = $450.00

Exterior Trim
- Paint fascia: 150 LF @ $3.00 = $450.00
- Paint exhaust vents: 2 EA @ $25.00 = $50.00

Subtotal: $14,450.00
Overhead & Profit (20%): $2,890.00
Total: $17,340.00
```

**Review:**

- Verify quantities match scope sheet
- Check unit costs are reasonable for your market
- Note any missing items

**Actions:**

- **Accept:** Proceed to upload carrier estimate
- **Regenerate:** If AI made an error, try again
- **Edit:** Manually adjust line items (coming soon)

### Step 4: Upload Carrier Estimate

When you receive the insurance company's estimate PDF:

1. Click "Upload Carrier Estimate"
2. Select the PDF file (max 10MB)
3. Click "Upload"
4. System parses PDF automatically (takes 1-2 minutes)

**What Happens:**

1. PDF is uploaded to secure storage
2. System extracts text from PDF
3. AI structures the data into line items
4. You're notified when parsing is complete

**If Parsing Fails:**

- Check PDF is text-based (not scanned image)
- Try re-uploading
- Contact support if issue persists

### Step 5: Compare Estimates

Once both estimates are ready:

1. Click "Compare Estimates"
2. System analyzes line-by-line differences (takes 30-45 seconds)
3. Review the comparison report

**What You'll See:**

```
Estimate Comparison

Summary:
- Industry Standard Total: $17,340.00
- Carrier Estimate Total: $14,650.00
- Difference: $2,690.00

Discrepancies Found: 2

1. Install new asphalt shingles
   Industry: $9,000.00 (2000 SF @ $4.50)
   Carrier: $7,000.00 (2000 SF @ $3.50)
   Difference: $2,000.00

   Justification: Carrier used outdated pricing. Current market
   rate for architectural shingles in Austin is $4.50/SF installed,
   per NRCA 2026 pricing guide.

2. Paint fascia
   Industry: $450.00 (150 LF @ $3.00)
   Carrier: $300.00 (150 LF @ $2.00)
   Difference: $150.00

   Justification: Carrier underestimated labor. Standard rate
   for exterior trim painting is $3.00/LF per local contractor
   surveys.
```

**Review:**

- Verify discrepancies are legitimate
- Check justifications are strong
- Note any items carrier omitted entirely

### Step 6: Generate Rebuttal Letter

When you're ready to challenge the carrier estimate:

1. Click "Generate Rebuttal Letter"
2. System creates professional business letter (takes 20-30 seconds)
3. Review the letter

**Sample Letter:**

```
February 6, 2026

To: Insurance Adjuster
[Carrier Name]

Re: Claim Review Request - Claim Number ABC123
Property Address: 123 Main Street, Austin, TX 78701
Policy Number: POL-123456

Dear Adjuster,

I am writing to request a reconsideration of the carrier estimate
provided for the property damage claim referenced above.

After careful review of the carrier estimate against current
industry-standard pricing for the Austin, TX market, we have
identified significant discrepancies that warrant discussion.

DISCREPANCIES IDENTIFIED:

1. Install new asphalt shingles
   Carrier Estimate: $7,000.00

   Difference: $2,000.00

   The carrier estimate appears to use outdated pricing for

   square foot, as documented in the National Roofing Contractors
   Association (NRCA) 2026 pricing guide...

[Full letter continues]

We respectfully request a meeting to discuss these discrepancies
and review the supporting documentation. We believe that adjusting
the estimate to reflect current market pricing is fair and
appropriate for all parties involved.

Thank you for your time and consideration. I look forward to
working with you to resolve this matter promptly.

Sincerely,

[Property Manager Name]
[Contact Information]
```

**Actions:**

- **Download as PDF:** Save letter to your computer
- **Print:** Print for mailing
- **Copy Text:** Copy to email or other document
- **Edit:** Make changes before sending (coming soon)

### Step 7: Follow Up

**After Sending Rebuttal:**

1. Track in claim notes
2. Follow up with carrier in 5-7 business days
3. Be prepared to provide:
   - Original scope sheet
   - Industry estimate details
   - Photos of damage
   - Local pricing documentation

**Possible Outcomes:**

- **Full Acceptance:** Carrier adjusts to match your estimate
- **Partial Acceptance:** Carrier increases some line items
- **Negotiation:** Further discussion needed
- **Denial:** Carrier maintains original estimate

### Best Practices

**Scope Sheet Quality:**

- Work with experienced contractors
- Verify measurements before accepting
- Include detailed notes about damage extent
- Take high-quality photos

**Estimate Review:**

- Always review AI-generated estimates
- Verify pricing matches your local market
- Adjust if needed before comparison
- Keep documentation of pricing sources

**Rebuttal Strategy:**

- Focus on strongest discrepancies first
- Provide clear justification
- Use professional, respectful tone
- Include supporting documentation
- Follow up promptly

**Documentation:**

- Save all versions of estimates
- Keep copies of rebuttal letters
- Track carrier responses
- Document all communications

## Troubleshooting

### Issue: Magic link doesn't work

**Symptoms:** Contractor clicks link and gets error page

**Solutions:**

1. Check link hasn't expired (7 days)
2. Verify contractor is using full URL (no truncation)
3. Try different browser
4. Clear browser cache
5. Resend new magic link

### Issue: Scope sheet won't submit

**Symptoms:** "Submit" button doesn't work or shows error

**Solutions:**

1. Check all required fields are filled
2. Verify internet connection
3. Try different browser
4. Check file size if photos attached
5. Contact support with claim ID

### Issue: PDF parsing failed

**Symptoms:** Parse status shows "failed" with error

**Common Causes:**

- PDF is image-based (scanned document)
- PDF is password-protected
- PDF is corrupted
- PDF format not supported

**Solutions:**

1. Re-scan as text-based PDF
2. Remove password protection
3. Try different PDF export settings
4. Contact support for manual parsing

### Issue: Generated estimate looks wrong

**Symptoms:** Line items missing or prices incorrect

**Solutions:**

1. Verify scope sheet data is accurate
2. Check your market area is set correctly
3. Regenerate estimate
4. Contact support with specific issues

### Issue: Comparison shows no discrepancies

**Symptoms:** Carrier estimate matches industry estimate

**This is normal if:**

- Carrier provided fair estimate
- Similar pricing sources used
- No underpayment occurred

**Next Steps:**

- Review both estimates manually
- Verify no items were omitted
- Consider this a positive outcome

## FAQ

### General

**Q: Is my data secure?**

A: Yes. All data is encrypted in transit and at rest. Access is restricted by authentication. PDFs are stored in secure cloud storage.

**Q: Who can see my claims?**

A: Only users in your organization. Each claim is isolated by organization ID in the database.

**Q: Can I delete a claim?**

A: Contact support for claim deletion. All associated data (scope sheets, estimates, letters) will be removed.

### Technical

**Q: What browsers are supported?**

A: Chrome, Firefox, Safari, and Edge (latest versions). Internet Explorer is not supported.

**Q: Is there a mobile app?**

A: Not yet. The web interface is mobile-responsive for basic viewing.

**Q: Can I integrate with other systems?**

A: API integration is available for enterprise customers. Contact support for details.

**Q: What's the uptime guarantee?**

A: 99.9% uptime SLA for enterprise customers. Standard customers: best effort.

### Billing

**Q: How is usage calculated?**

A: Per AI operation (estimate, comparison, rebuttal). Typically 3 operations per claim.

**Q: What if I exceed my plan?**

A: You'll receive an email notification. Overage is billed at standard rate.

**Q: Can I get a refund?**

A: Contact support to discuss specific situations.
